# AI Safety Research Workflow

When this skill is triggered, invoke the `/researcher` command to run the full end-to-end research process. If the user has provided a specific topic, pass it as the argument.

This plugin provides a structured multi-agent research workflow for AI safety R&D, including:
- Literature discovery and synthesis
- Gap analysis and hypothesis generation
- Experimental design
- Critical review and iteration
- Report generation
